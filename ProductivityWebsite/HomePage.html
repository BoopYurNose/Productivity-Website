Welcome to my site

This is a site, that I made as a tool to track my productivity. Regularly I will journal daily,
and write down a list of all the tasks I want to get done everyday in a journal.
I wanted to make a website where I can:
  1. Log what I'm doing everyday (like a blog)
  2. Allow me to add items to a to-do list
  3. Allows me to tick off items on a to-do list and changes my productivity score depending on how many I ticked off
  4. Rates my productivity score (percentage based on how many tasks I got done that day)
  5. Track sleep, Coding time, Drawing time, etc
  6. show progress chart overtime using data from previous days
